# iOS（Testflight安装）

－简单 无需注册，无需设置，一键翻墙。 下载图灵VPN后点击开关就可翻墙。

－快速 不需要再继续等待境外网页的缓慢加载。使用图灵VPN加速境外网页浏览。

－免费 让每个人都能自由访问互联网。

－软件轻量 图灵VPN只占7MB的空间。

使用方式：

1 [点击进入Testflight链接](https://testflight.apple.com/join/ud1FVIMV) 

2 打开链接后，按照步骤首先点击安装Testflight

3 按照步骤点击安装图灵vpn

4 在Testflight中设置打开自动更新
